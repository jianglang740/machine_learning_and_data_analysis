# 删除数据

## 一、删除 **行**（最常用）
### 1. 根据 **索引** 删除行
```python
df.drop(index=0)         # 删除第 0 行
df.drop(index=[1,3,5])   # 删除多行
df.drop(index=df.index[[0,2]]) # 按位置删
```

### 2. 根据 **条件** 删除行
```python
df = df[df['age'] > 18]    # 保留age>18，相当于删除不符合条件的
df = df[~(df['name'] == '张三')]  # 删除name=张三的行
```

### 3. 删除 **重复行**
```python
df.drop_duplicates()          # 全列重复才删
df.drop_duplicates(['name']) # 只看name列，重复就删
df.drop_duplicates(keep='first') # 保留第一个
df.drop_duplicates(keep='last')  # 保留最后一个
```

### 4. 删除 **空值行**
```python
df.dropna()               # 只要有一个NaN就删除整行
df.dropna(how='all')      # 整行全是NaN才删除
df.dropna(subset=['age']) # age列有空值就删除
```

---

## 二、删除 **列**
### 1. 根据列名删除列
```python
df.drop(columns='age')          # 删除age列
df.drop(columns=['name', 'sex'])# 删除多列
```

### 2. 直接删除列（简洁写法）
```python
del df['age']
```

---

## 三、高级删除
### 1. 删除后 **直接修改原表**
```python
df.drop(columns='age', inplace=True)
```

### 2. 删除 **索引名/列名** 包含特定字符
```python
df.drop(columns=[col for col in df.columns if 'test' in col])
```

---

## 四、最常用总结（背这几句就够）
```python
# 删除行
df.drop(index=0)
df.drop(index=[1,2])
df = df[df['age']>18]
df.drop_duplicates()
df.dropna()

# 删除列
df.drop(columns='age')
del df['age']
```
