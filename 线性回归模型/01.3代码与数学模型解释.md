## 一、核心逻辑对应（代码=OLS理论）
### 1. 数据提取（对应总体/样本模型）
```python
x = df["s"].values.reshape(-1, 1)  # 教育年限s（横轴，解释变量x）
y = df["lnw"].values                # 工资对数lnw（纵轴，被解释变量y）
```
- 对应理论：$y_i = \alpha + \beta x_i + \varepsilon_i$（总体模型），$y_i = \hat{\alpha} + \hat{\beta}x_i + e_i$（样本模型）
- 这里`x`就是教育年限$s$，`y`就是工资对数$\ln w$，和你之前的分析完全一致。

### 2. OLS拟合（对应最小二乘法求参数）
```python
model = LinearRegression()
model.fit(x, y)
y_pred = model.predict(x)
```
- `LinearRegression()`：sklearn的线性回归类，**底层就是用最小二乘法（OLS）求解$\hat{\alpha}$和$\hat{\beta}$**
- `model.fit(x, y)`：执行OLS拟合，计算出最优的截距$\hat{\alpha}$和斜率$\hat{\beta}$
- `y_pred = model.predict(x)`：用拟合出的参数计算**拟合值$\hat{y}_i = \hat{\alpha} + \hat{\beta}x_i$**，对应你理论里的样本回归线

### 3. 提取OLS核心参数（对应推导结果）
```python
slope = model.coef_[0]       # 斜率$\hat{\beta}$（教育回报率）
intercept = model.intercept_ # 截距$\hat{\alpha}$
r_squared = model.score(x, y)# 决定系数$R^2$
```
- 对应你推导的最终结果：
  - $\hat{\beta} = \frac{\sum (x_i-\bar{x})(y_i-\bar{y})}{\sum (x_i-\bar{x})^2}$ → 代码里`slope`，你运行结果是`0.0966`（多上1年学，工资对数涨0.0966，即工资涨约9.7%，完全符合计量结论）
  - $\hat{\alpha} = \bar{y} - \hat{\beta}\bar{x}$ → 代码里`intercept`，结果`4.3915`
  - $R^2 = \frac{ESS}{TSS}$ → 代码里`r_squared`，结果`0.2527`（教育解释了25.27%的工资波动，和你理论里的平方和分解完全对应）

## 三、补充：代码底层的OLS原理
`sklearn.linear_model.LinearRegression`的`fit()`方法，**完全就是你之前学的OLS推导**：
1.  定义残差平方和目标函数：$Q = \sum (y_i - \hat{\alpha} - \hat{\beta}x_i)^2$
2.  对$\hat{\alpha}$、$\hat{\beta}$求偏导，令偏导为0，得到正规方程组
3.  解方程组，得到$\hat{\alpha}$和$\hat{\beta}$的闭式解（就是你推导的公式）
4.  计算$R^2$，衡量拟合优度
