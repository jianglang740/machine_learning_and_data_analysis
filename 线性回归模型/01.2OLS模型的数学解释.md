## 一、先明确两个核心模型（分清总体和样本）
### 1. 总体回归模型（真实世界，我们看不到）
真实的工资-教育关系是：
$$y_i = \alpha + \beta x_i + \varepsilon_i$$
- $y_i$：第$i$个人的真实工资对数（$\ln w$）
- $x_i$：第$i$个人的教育年限（$s$）
- $\alpha, \beta$：**真实参数**（上帝视角，我们永远不知道准确值）
- $\varepsilon_i$：**随机扰动项**（能力、运气、家庭背景等没观测到的因素）

### 2. 样本回归模型（用数据算出来的，我们能拿到）
用样本数据拟合出的「猜测模型」：
$$y_i = \hat{\alpha} + \hat{\beta} x_i + e_i$$
- $\hat{\alpha}, \hat{\beta}$：**样本估计量**（用你手里的758个数据算出来的）
- $e_i = y_i - \hat{y}_i$：**残差**（真实值$y_i$ - 拟合值$\hat{y}_i = \hat{\alpha} + \hat{\beta}x_i$，就是模型没猜准的部分）

---

## 二、OLS目标函数：为什么要最小化残差平方和？
我们的目标是：让「猜测的直线」尽可能贴近真实数据，也就是让**残差$e_i$尽可能小**。
但直接加总残差$\sum e_i$会正负抵消，所以用**残差平方和**衡量总误差：
$$Q = \sum_{i=1}^n e_i^2 = \sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i)^2$$
我们要找$\hat{\alpha}, \hat{\beta}$，让$Q$最小，这就是**普通最小二乘法（OLS）**的核心。

---

## 三、第一步：对目标函数求偏导（你之前卡的部分，再推一遍）
对多元函数求最小值，微积分的方法是：**对每个自变量求偏导，令偏导数=0**（极值点的必要条件）。
这里的自变量是$\hat{\alpha}$和$\hat{\beta}$，$y_i, x_i$是样本常数，求导时只对$\hat{\alpha}, \hat{\beta}$求导。

### 1. 对$\hat{\alpha}$求偏导
用**复合函数求导法则**：$(u^2)' = 2u \cdot u'$，令$u = y_i - \hat{\alpha} - \hat{\beta}x_i$
- 外层求导：$2u$
- 内层对$\hat{\alpha}$求导：$\frac{\partial u}{\partial \hat{\alpha}} = -1$（$y_i, x_i, \hat{\beta}$都是常数，导数为0）
- 单样本偏导：$2(y_i - \hat{\alpha} - \hat{\beta}x_i) \cdot (-1) = -2(y_i - \hat{\alpha} - \hat{\beta}x_i)$
- 对所有样本求和：
$$\frac{\partial Q}{\partial \hat{\alpha}} = -2\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i)$$
令偏导数=0：
$$-2\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i) = 0 \tag{1}$$

### 2. 对$\hat{\beta}$求偏导
同样用复合函数求导：
- 内层对$\hat{\beta}$求导：$\frac{\partial u}{\partial \hat{\beta}} = -x_i$（$y_i, \hat{\alpha}$是常数，导数为0）
- 单样本偏导：$2(y_i - \hat{\alpha} - \hat{\beta}x_i) \cdot (-x_i) = -2(y_i - \hat{\alpha} - \hat{\beta}x_i)x_i$
- 对所有样本求和：
$$\frac{\partial Q}{\partial \hat{\beta}} = -2\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i)x_i$$
令偏导数=0：
$$-2\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i)x_i = 0 \tag{2}$$

### 3. 约去-2，得到正规方程组（你PPT的公式0.8）
因为$-2 \neq 0$，等式两边同时除以-2，得到：
$$
\begin{cases}
\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i) = 0 \\
\sum_{i=1}^n (y_i - \hat{\alpha} - \hat{\beta}x_i)x_i = 0
\end{cases}
\tag{3}
$$
这就是求解$\hat{\alpha}, \hat{\beta}$的核心方程组。

---

## 四、第二步：解正规方程组，得到样本回归线
### 1. 先化简第一个方程
把求和拆开：
$$\sum_{i=1}^n y_i - \sum_{i=1}^n \hat{\alpha} - \sum_{i=1}^n \hat{\beta}x_i = 0$$
- $\sum_{i=1}^n \hat{\alpha} = n\hat{\alpha}$（$\hat{\alpha}$是常数，$n$个样本求和）
- $\sum_{i=1}^n \hat{\beta}x_i = \hat{\beta}\sum_{i=1}^n x_i$（$\hat{\beta}$是常数）
- 样本均值：$\bar{y} = \frac{1}{n}\sum y_i$，$\bar{x} = \frac{1}{n}\sum x_i$，即$\sum y_i = n\bar{y}$，$\sum x_i = n\bar{x}$

代入得：
$$n\bar{y} - n\hat{\alpha} - \hat{\beta} \cdot n\bar{x} = 0$$
两边除以$n$：
$$\bar{y} - \hat{\alpha} - \hat{\beta}\bar{x} = 0$$
整理出$\hat{\alpha}$：
$$\hat{\alpha} = \bar{y} - \hat{\beta}\bar{x} \tag{4}$$
**结论1：样本回归线一定过样本均值点$(\bar{x}, \bar{y})$**

### 2. 化简第二个方程，求$\hat{\beta}$
把$e_i = y_i - \hat{\alpha} - \hat{\beta}x_i$代入第二个方程，得：
$$\sum_{i=1}^n e_i x_i = 0$$
再把$\hat{\alpha} = \bar{y} - \hat{\beta}\bar{x}$代入$e_i$：
$$e_i = y_i - (\bar{y} - \hat{\beta}\bar{x}) - \hat{\beta}x_i = (y_i - \bar{y}) - \hat{\beta}(x_i - \bar{x})$$
代入$\sum e_i x_i = 0$：
$$\sum_{i=1}^n \left[(y_i - \bar{y}) - \hat{\beta}(x_i - \bar{x})\right]x_i = 0$$
拆开求和：
$$\sum (y_i - \bar{y})x_i - \hat{\beta}\sum (x_i - \bar{x})x_i = 0 \tag{5}$$

#### 关键化简：用离均差形式简化
定义**离均差**：$y_i^* = y_i - \bar{y}$，$x_i^* = x_i - \bar{x}$（每个样本减去均值，消除截距影响）
- 性质：$\sum x_i^* = \sum (x_i - \bar{x}) = 0$，$\sum y_i^* = 0$
- 化简$\sum (x_i - \bar{x})x_i = \sum x_i^* (x_i + \bar{x} - \bar{x}) = \sum x_i^{*2}$（因为$\sum x_i^* \bar{x} = \bar{x}\sum x_i^* = 0$）
- 化简$\sum (y_i - \bar{y})x_i = \sum y_i^* (x_i + \bar{x}) = \sum y_i^* x_i^*$（同理$\sum y_i^* \bar{x} = 0$）

代入式(5)：
$$\sum_{i=1}^n x_i^* y_i^* - \hat{\beta}\sum_{i=1}^n x_i^{*2} = 0$$
解出$\hat{\beta}$：
$$\hat{\beta} = \frac{\sum_{i=1}^n x_i^* y_i^*}{\sum_{i=1}^n x_i^{*2}} = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2} \tag{6}$$
这就是**斜率的OLS估计量**，也是你最熟悉的公式。

### 3. 最终样本回归线
把式(6)代入式(4)，得到截距$\hat{\alpha}$，最终样本回归线为：
$$\hat{y}_i = \hat{\alpha} + \hat{\beta}x_i = (\bar{y} - \hat{\beta}\bar{x}) + \hat{\beta}x_i = \bar{y} + \hat{\beta}(x_i - \bar{x})$$
到这里，你就完全看懂「求出样本回归线」的全过程了✅

## 五、第三步：正交性是什么？为什么重要？
### 1. 正交的直观含义
**正交 = 垂直 = 不相关 = 信息完全提取**
在回归里，正交性就是从正规方程组直接推出来的两个结论：
$$
\begin{cases}
\sum_{i=1}^n e_i = 0 \quad \text{（残差和为0）} \\
\sum_{i=1}^n e_i x_i = 0 \quad \text{（残差与解释变量正交）}
\end{cases}
$$

### 2. 正交性的经济含义
- $\sum e_i = 0$：残差的平均值为0，模型没有系统性偏差（不会整体高估/低估工资）
- $\sum e_i x_i = 0$：**残差和教育年限$x_i$不相关**，意味着「教育年限的信息已经被回归线完全提取了，残差里再也没有教育的影响了」

### 3. 正交性的几何意义
把$y, x, e$看作$n$维空间的向量：
- $y$是真实值向量，$\hat{y}$是拟合值向量，$e$是残差向量
- $y = \hat{y} + e$
- 正交性：$e \perp x$，$e \perp \hat{y}$（残差向量和解释变量、拟合值向量都垂直）
- 几何上：$\hat{y}$是$y$在$x$张成的空间上的**投影**，OLS就是找这个投影的过程

## 六、第四步：平方和分解（R²的由来）
### 1. 总平方和分解
用离均差形式，把真实值的波动拆成「可解释部分+不可解释部分」：
$$y_i - \bar{y} = (\hat{y}_i - \bar{y}) + (y_i - \hat{y}_i) = (\hat{y}_i - \bar{y}) + e_i$$
两边平方后求和：
$$\sum (y_i - \bar{y})^2 = \sum (\hat{y}_i - \bar{y})^2 + \sum e_i^2 + 2\sum (\hat{y}_i - \bar{y})e_i$$
**关键：交叉项$\sum (\hat{y}_i - \bar{y})e_i = 0$**（由正交性$\sum e_i x_i = 0$推导，$\hat{y}_i$是$x_i$的线性组合，所以和$e_i$正交）

因此得到**平方和分解公式**：
$$\underbrace{\sum (y_i - \bar{y})^2}_{TSS} = \underbrace{\sum (\hat{y}_i - \bar{y})^2}_{ESS} + \underbrace{\sum e_i^2}_{RSS}$$
- $TSS$（总平方和）：$y$的总波动（工资本身的差异）
- $ESS$（解释平方和）：被$x$解释的波动（教育解释的工资差异）
- $RSS$（残差平方和）：未被解释的波动（能力、运气等其他因素）

### 2. 拟合优度$R^2$
$R^2$衡量「模型解释了多少总波动」：
$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}$$
- $R^2 \in [0,1]$，越接近1，模型拟合越好
- 你用grilic数据算出来的$R^2≈0.25$，就是「教育年限解释了25%的工资差异」

---

## 七、最后：线性回归到底在做什么？（一句话总结）
**线性回归是用样本数据，通过最小化残差平方和，找到一条最贴合数据的直线（样本回归线），用它来估计真实世界的总体规律；正交性保证估计无偏，平方和分解衡量模型的解释力度。**
