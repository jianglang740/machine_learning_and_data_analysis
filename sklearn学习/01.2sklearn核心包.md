下面这份是**sklearn 官方结构整理版**，按**模块包名 → 里面有什么估计器/函数 → 用途**分类，你以后查 API、记结构、写代码直接对照这一份就够了。

# 一、sklearn 核心包结构总览
```
sklearn
├── linear_model       线性模型
├── tree               树模型
├── ensemble           集成模型
├── neighbors          近邻
├── svm                支持向量机
├── naive_bayes        朴素贝叶斯
├── preprocessing      预处理、变换
├── impute             缺失值填充
├── decomposition      降维
├── feature_selection  特征选择
├── cluster            聚类
├── model_selection    模型选择、划分、CV
├── metrics            评估指标
├── datasets           数据集
├── pipeline           流水线
└── utils              工具
```

# 二、各模块详细整理

## 1. sklearn.linear_model 线性模型（估计器 = 类）
**分类：**
- LogisticRegression 逻辑回归
- RidgeClassifier 岭分类
- SGDClassifier 随机梯度下降分类

**回归：**
- LinearRegression 线性回归
- Ridge 岭回归
- Lasso L1 正则
- ElasticNet 弹性网
- SGDRegressor 梯度下降回归

**通用方法：**
- `.fit(X, y)`
- `.predict(X)`
- `.score(X, y)`
- `.coef_ / .intercept_` 属性

---

## 2. sklearn.tree 树模型
- DecisionTreeClassifier 分类树
- DecisionTreeRegressor 回归树

方法同上：fit / predict / score

---

## 3. sklearn.ensemble 集成模型
**分类：**
- RandomForestClassifier
- GradientBoostingClassifier
- AdaBoostClassifier
- HistGradientBoostingClassifier

**回归：**
- RandomForestRegressor
- GradientBoostingRegressor
- AdaBoostRegressor

---

## 4. sklearn.neighbors 近邻
**分类：**
- KNeighborsClassifier

**回归：**
- KNeighborsRegressor

**无监督：**
- NearestNeighbors

---

## 5. sklearn.svm
**分类：**
- SVC
- LinearSVC

**回归：**
- SVR
- LinearSVR

---

## 6. sklearn.naive_bayes 朴素贝叶斯
- GaussianNB
- MultinomialNB
- BernoulliNB

---

# 三、预处理 & 特征工程（全是估计器，可进 Pipeline）

## 1. sklearn.preprocessing
**标准化/归一化：**
- StandardScaler
- MinMaxScaler
- RobustScaler
- MaxAbsScaler

**编码：**
- OneHotEncoder
- OrdinalEncoder
- LabelEncoder（用于 y）

**变换：**
- PolynomialFeatures 多项式
- KBinsDiscretizer 分桶
- FunctionTransformer 自定义变换
- Binarizer 二值化

**通用方法：**
- `.fit(X)`
- `.transform(X)`
- `.fit_transform(X)`

---

## 2. sklearn.impute 缺失值填充
- SimpleImputer 均值/中位数/众数填充
- KNNImputer KNN 填充

---

## 3. sklearn.decomposition 降维
- PCA
- TruncatedSVD
- NMF
- FastICA

方法：fit / transform

---

## 4. sklearn.feature_selection 特征选择
- VarianceThreshold 方差过滤
- SelectKBest 单变量筛选
- SelectFromModel 基于模型筛选
- RFE 递归特征消除

---

## 5. sklearn.cluster 聚类（无监督估计器）
- KMeans
- DBSCAN
- AgglomerativeClustering
- GaussianMixture (GMM)

方法：
- `.fit(X)`
- `.predict(X)`
- `.fit_predict(X)`

---

# 四、model_selection（一半类，一半函数）
## 属于**函数**（不是估计器）
- train_test_split 划分数据集

## 属于**类（估计器）**
- KFold
- StratifiedKFold
- GroupKFold
- TimeSeriesSplit
- GridSearchCV 网格搜索
- RandomizedSearchCV 随机搜索

GridSearchCV 等本身也是估计器，可以 fit/predict。

# 八、一张表帮你彻底区分“估计器 vs 函数”
| 特点 | 估计器（Class） | 工具函数（Function） |
|------|------------------|------------------------|
| 写法 | model = LR() | train_test_split() |
| .fit() | 有 | 无 |
| 保存状态 | 有（均值、权重等） | 无 |
| 可放入Pipeline | 可以 | 不可以 |
| 例子 | Scaler、PCA、模型 | 划分、指标、造数据 |
